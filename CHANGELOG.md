# Changelog

All notable changes to the **YouDownloader03** project will be documented in this file.

## [1.0.0] - 2026-04-21

### Added
- Official project launch.
- Support for Single Video downloads (MP4, MP3).
- Support for Playlist downloads with bulk processing.
- Real-time download progress tracking using Server-Sent Events (SSE).
- ZIP archiving for playlist downloads.
- Professional dark-themed UI with Tailwind CSS.
- Download history with persistence.

### Changed
- Rebranded from "YouTube Downloader Pro" to **YouDownloader03**.
- Updated font style to Bold Normal for cleaner typography.
- Refined logo and visual identity with custom SVG iconography.
- Switched project license from MIT to **Apache License 2.0** for enhanced legal and patent protection.

### Fixed
- Improved error handling for invalid YouTube URLs.
- Sanitized filenames to prevent issues with special characters.
- Fixed CORS issues for video metadata fetching.
