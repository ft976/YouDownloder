# Security Policy

## Supported Versions

We only support the latest version of YouDownloader03. Please ensure you are running the most recent update before reporting a security vulnerability.

| Version | Supported          |
| ------- | ------------------ |
| v1.0.x  | ✅ Supported        |
| < v1.0  | ❌ Not Supported    |

## Reporting a Vulnerability

We take the security of our users seriously. If you find a security vulnerability, please do NOT create a public issue. Instead, please report it via the project's security reporting channel.

Once we receive your report, we will:
1. Acknowledge the report within 48 hours.
2. Investigate the issue and evaluate the impact.
3. Provide an estimated timeline for a fix.
4. Notify you once the fix is deployed.

Thank you for helping keep YouDownloader03 secure!
